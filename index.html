TI nas Escolas Públicas

Projeto "TI nas Escolas Públicas"

Idealizado por Clara Monaco, este projeto tem como objeto atender ás demandas do mercado em constante crescimento do mercado de Tecnologia da Informação. Além disso, busca promover a inclusão de adolescentes do ensino médios de escolas públicas e estaduais, fortalecendo aqueles que têm menos oportunidades, estimulando o desenvolvimento de novos conhecimentos e interesses, e abrindo caminhos para futuras oportunidades. Concluindo, este projeto é uma inciativa importante que não apenas supre as demandas do mercado de TI, mas também empodera e capacita os adolescentes de escolas públicas e estaduais, proporcionando novas perspectivas e abrindo portas para um futuro promissor.

Oficinas do Projeto

  • Logica e Programação
  • Desenvolvimento Web front End
  • JavaScript

Objetivo do Projeto

Projeto de TI nas escolas públicas capacita alunos e professores para o futuro O objetivo é promover aulas interativas com acesso a tecnologia e estimular habilidades digitais. O projeto busca oferecer oportunidades diferentes para os alunos e capacitar a nova geração para o mundo digital.

Importância da TI nas escolas

TI nas escolas é imporante porque ajuda a melhorar a qualidade da educação, facilitando o acesso á informação, promove a interação e colaboração entre os alunos e professores, também prepara os estudantes para o mundo digital

Nossa equipe

A equipe será composta por um grupo de professores, que receberam suas oficinas para a sua aerea especifica

Logica e Programação

O objetivo dessa oficina é capacitar os participantes a desenvolver algoritimos bem estruturados utilizando princípios de lógica de programação. Ele é direcionado a indivíduos interessados em adquirir conhecimentos para criar programas computacionais.

Desenvolvimento Web front End

Este curso tem como objetivo capacitar as alunas com as habilidades e conhecimentos necessários para projetar e desenvolver a interface do usuário de um site ou aplicativa web

JavaScript

Este curson tem como objetivo capacitar as alujnas com as habilidades e conhecimentos necessários para adicionar interatividade e funcionalidade em páginas da web utilizando a linguagem de programação JavaScript.
